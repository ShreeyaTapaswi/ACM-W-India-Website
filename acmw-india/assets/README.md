# ACM-W India Website — Assets Folder

Place your images here following this structure:

```
assets/
├── acmw-india-logo.png       ← ACM-W India logo (main logo, ~200x70px)
├── banner.jpg                ← Hero/banner background image (1920x600px recommended)
├── placeholder.png           ← Auto-used when image is missing (already included)
│
├── news/
│   ├── event1.jpg            ← Used on homepage "Annual Summit" news card
│   ├── newsletter.jpg        ← Used on homepage "Newsletter" news card
│   └── chapter.jpg           ← Used on homepage "Chapter Launch" news card
│
└── newsletters/
    ├── may-2026.png          ← Newsletter cover thumbnail
    ├── apr-2026.png
    ├── mar-2026.png
    ├── feb-2026.png
    ├── jan-2026.png
    ├── dec-2025.png
    ├── nov-2025.png
    ├── oct-2025.png
    ├── sep-2025.png
    ├── aug-2025.png
    ├── jul-2025.png
    └── jun-2025.png
```

## Tips
- Newsletter covers work best as portrait/tall images (e.g., 300×400px)
- News card images should be landscape (e.g., 600×340px, 16:9 ratio)
- The logo can be PNG with transparent background
- If an image is missing, the site will automatically use `placeholder.png`
